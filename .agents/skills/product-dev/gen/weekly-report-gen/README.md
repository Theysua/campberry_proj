# 周报生成器 (weekly-report-gen)

## 概述

从 git 日志、任务列表或用户笔记自动生成结构化周报。

## 使用场景

- 每周结束需要写状态报告
- 从 git 历史和任务跟踪器汇总工作
- 创建团队进度更新

## 快速开始

```
帮我写周报，这周做了用户模块重构和 3 个 bug 修复
```

## 输入

| 参数 | 必需 | 说明 |
|------|------|------|
| git_log / task_list / notes | 至少一个 | 工作数据来源 |
| date_range | 否 | 周期范围（默认最近 7 天） |
| author | 否 | git 作者过滤 |
| language | 否 | 输出语言（默认中文） |

## 输出

周报文档：本周完成、进行中、遇到的问题、下周计划
