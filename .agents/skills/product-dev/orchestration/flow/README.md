# 工作流引擎 (flow)

## 概述

执行预定义或自定义的文档工作流，将多个 skill 按逻辑顺序串联执行。

## 使用场景

- 需要运行标准的多步文档流水线（BRD → PRD → TRD → 测试）
- 自动化可重复的文档生成流程
- 为新功能创建完整文档套件

## 预定义工作流

| 工作流 | 说明 | 步骤数 |
|--------|------|--------|
| `new-feature` | 完整功能文档流水线 | 8 步 |
| `quick-spec` | 轻量级规格（跳过 BRD） | 4 步 |
| `api-first` | API 驱动开发 | 4 步 |
| `decision-record` | 架构决策 | 3 步 |

## 快速开始

```
运行 new-feature 工作流，功能是用户通知系统
```

## 关联 Skill

- `project-init` — 先初始化项目结构，再运行工作流
- `iteration-coordinator` — 多文档联动迭代
